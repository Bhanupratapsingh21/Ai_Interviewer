AI Interviewer is an AI-first interviewing platform where companies create interviews and candidates participate in real-time voice interviews with an AI interviewer.

- Public & invite-only interviews
- Resume-aware questioning
- Automatic evaluation
- Docker self-hosting
- SaaS deployment

# Rules for Coding Agents

- Never bypass architecture.
- Never put Prisma inside controllers.
- Never skip validation.
- Prefer reusable services.
- Keep modules isolated.
- Build backend before AI.
- Keep Docker compatible.
- Maintain Bun + Turborepo compatibility.
- Preserve Prisma 6.16.2 unless explicitly upgraded.
- Prefer REST APIs.
- Keep code production-ready.

This document should be treated as the primary context for future development.
